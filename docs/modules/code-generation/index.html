Code Generation

Code Generation

Work with limitless Personally Identifiable Information (PII) in the most secure way.

Overview

A Beta version is currently available to generate code out of a schema.

At this moment, only Hibernate Validation (Java 8+) is supported.

Basic usage

./schematix generate-source --generator=hibernate-validator example.schema.json

Hibernate Validator

Schematix can generate a basic validator in Java using the following command:

./schematix generate-source --generator=hibernate-validator example.schema.json

PySpark

Schematix can generate a basic reader for PySpark too:

Using the following command:

./schematix generate-source --generator=pyspark --mapping=example.mapping.json example.schema.json

Schematix will generate the following:

df = spark.read.text("/tmp/sample.txt")
df.select(
    df.value.substr(1,3).alias('id'),
    df.value.substr(4,8).alias('date'),
    df.value.substr(12,3).alias('string'),
    df.value.substr(15,4).cast('integer').alias('integer')
).show()
